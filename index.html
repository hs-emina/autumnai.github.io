From 2020-09-12 7am UTC to 2020-09-19 7am UTC

A seven-day programming contest organized with the collaboration of CodinGame, dedicated to motivate Islington students practice Artificial Intelligence (AI) using Codingame's interface. Even though, only Islington students qualify for prizes, everyone is invited to participate and have fun.

The game for the contest is Code of Kutulu. You control your explorer (character) and compete with three other players in a set of pre-generated maps. The goal is to write an Artificial Intelligence (AI) for Code of Kutulu that is able to survive the longest.
